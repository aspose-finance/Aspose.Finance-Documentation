Vi är glada att kunna tillkännage den första offentliga utgåvan av Aspose.Finance for .NET. Aspose.Finance for .NET har rika funktioner

- Skapa XBRL-instans från början
- Läs XBRL format
- Läs iXBRL format
- XBRL validering
- iXBRL validering
